## 회원 가입

| Actor Action                                                                 | System Response          |
| ---------------------------------------------------------------------------- | ------------------------ |
| 1. ID, 비밀번호, 전화번호, 결제 수단, 선호 자전거 유형(일반/전기) 등의 정보를 입력한다. | 2. 모든 정보가 제대로 입력되었는지 확인 후 회원 가입 완료 메시지를 제공한다.|

| Actor Action                      | System Response          |
| --------------------------------- | ------------------------ |
| 1. 회원 탈퇴 버튼을 누른다. | 2. 회원 탈퇴에 대해 다시 한 번 확인하는 팝업을 띄운다. |
| 3. 확인 버튼을 누른다. | 4. 회원 탈퇴 확인 메시지를 제공한다. |

## 대여소 상세정보 보기 및 자전거 대여/예약대기 신청

| Actor Action        | System Response          |
| ------------------- | ------------------------ |
| 1. None             | 2. 대여소 리스트 화면을 표시 |
| 3. 회원은 특정 대여소를 선택 | 4. 회원이 조회한 대여소의 상세정보화면을 출력 (대여소 이름, 위치 사용 가능 자전거 목록 등이 출력됨)    |
| 5.	회원은 자전거 대여 혹은 자전거 예약대기 신청을 함 | 6.	회원이 자전거 대여 혹은 자전거 예약대기 신청 시 문자로 알림을 보냄|

## 자전거 대여 정보 조회

| Actor Action        | System Response          |
| ------------------- | ------------------------ |
| 1. None | 2. 자전거 대여 정보 조회 화면을 표시 |
| 3. 회원은 현재 대여 중인 자전거를 조회 | 4. 회원이 조회한 자전거 리스트를 출력 (대여소 이름, 대여소 위치, 자전거 ID, 자전거 제품명, 자전거 유형이 출력됨)|

## 자전거 예약대기 정보 조회/취소

| Actor Action        | System Response          |
| ------------------- | ------------------------ |
| 1. None | 2. 자전거 예약대기 정보 조회 화면을 표시 |
| 3. 회원은 자전거 예약대기 정보를 조회 (대여소 이름, 대여소 위치, 자전거 ID, 자전거 제품명, 자전거 유형등을 입력 후 조회) | 4.	회원이 조회한 자전거에 해당하는 예약대기 리스트를 출력|
| Extension : 4단계 이후, 회원은 원하는 예약대기를 취소 할 수 있다. |

## 자전거 반납
| Actor Action        | System Response          |
| ------------------- | ------------------------ |
| 1. None | 2. 자전거 대여 정보 조회 화면을 표시|
| 3. 회원은 특정 자전거를 지정된 대여소에 반납||	
| Extension : 3단계 이후, 회원은 식당 예약 서비스와 연결할 수 있다. <br> 3단계 이후, 반납된 자전거의 대기 예약 1순위 회원에게 예약되었다는 이메일을 보낸다. |
